#  Simple-Web-Server

 # Project Files
`server.py`: Contains the Python application code** that sets up and runs the web server. It handles incoming HTTP requests.
* **`index.html`**: The **frontend file** served by the server. It includes the HTML structure and styling (CSS) to make the page responsive and interactive.

 # To Run the Server

# Prerequisites
You only need Python installed on your system.

# Steps

Ensure all Files are in the Same Directory
Execute the Script: Open your terminal or command prompt in the project directory and run:
python server.py
Access the Website: The server will start listening on port `8080`. Open your web browser and navigate to the following URL:
    ```
    [http://127.0.0.1:8080](http://127.0.0.1:8080)
    ```
