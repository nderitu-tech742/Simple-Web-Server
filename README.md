#  Simple-Web-Server

 # Project Files
`server.py`: Contains the Python application code** that sets up and runs the web server. It handles incoming HTTP requests.
* **`index.html`**: The **frontend file** served by the server. It includes the HTML structure and styling (CSS) to make the page responsive and interactive.

 # To Run the Server

# Prerequisites
You only need Python installed on your system.

# Steps

1. **Ensure all files are in the same directory**

2. **Execute the script**  
   Open your terminal or command prompt in the project directory and run:  
   ```bash
   python server.py

3. **Access the website**
   The server will start listening on port 8080.
    Open your web browser and navigate to: http://127.0.0.1:8080
